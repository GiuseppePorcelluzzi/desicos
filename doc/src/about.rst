Installation
------------

**1) Downloading**

The source code repository can be downloaded from
`GitHub <https://github.com/desicos/desicos/archive/master.zip>`_
or cloned using `Git <http://git-scm.com/downloads>`_::

    $ git clone https://github.com/desicos/desicos.git

You can also
`visit the source code page here <https://github.com/desicos/desicos>`_.

Requirements
------------
- numpy
- scipy
